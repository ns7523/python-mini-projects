### Stock APP

This App show stock details of the entered ticker.
	-	Compnay Profile
	-   	Income Statement Details
	-	EPS Chart
	-	Company Current Quotes
	
Modules Used:
	- Financial Modelling Perp API
	- requests
	- Flask
