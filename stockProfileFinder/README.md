### Stock Profile Finder - CLI Project

Stock details finder project in Python. 
Modules Used
  - Pandas
  - Plotly
  - Requests
  - Financial Modelling API

User can find various stock information:
  - Income Statment
  - Company Ratios
  - Company Profile
  - Growth Ratios
  - Qutoes
  - Stock Prices with graphs
  - Stock Dividends
