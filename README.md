## 06 · Installation

```bash
git clone https://github.com/ns7523/Python-mini-projects.git
cd Python-mini-projects
python -m venv .venv
source .venv/bin/activate
pip install notebook pandas numpy flask requests beautifulsoup4
```

---

## 07 · Usage

Run Python scripts:

```bash
python path/to/script.py
```

Run Flask projects:

```bash
python app.py
```

Launch notebooks:

```bash
jupyter notebook
```
